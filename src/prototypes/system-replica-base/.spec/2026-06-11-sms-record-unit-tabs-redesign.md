# 2026-06-11 短信发送记录查询页改版决策快照

## 用户诉求

将短信管理模块的“短信发送记录查询”页面做改版：当前页面设定为“本单位”页签，新增“其他单位”页签；其他单位内容与当前页面接近，但需要新增“发送单位”字段，字段值包含大竹县、达川区等单位。

## 已确认方案

- 页面仍延展在 `src/prototypes/system-replica-base/` 底座内，不新建原型。
- 页面标题保持“短信发送记录查询”。
- 查询区上方增加单位范围页签：`本单位`、`其他单位`。
- 默认展示 `本单位` 页签，沿用原有查询字段和表格结构。
- `其他单位` 页签新增查询字段 `发送单位`，示例选项为：大竹县、达川区、宣汉县、开江县、渠县、万源市。
- `其他单位` 表格新增 `发送单位` 列，放在序号之后，便于快速识别短信来源。
- 查询、重置、分页、表格等仍复用现有蓝灰色后台样式和底座组件。

## 设计取舍

本次不做营销化或大屏化重设计，不改变短信管理模块整体导航结构；重点改善同一页面内“本单位/其他单位”的范围区分和发送来源识别。

## 验收重点

- 打开短信发送记录查询页时，默认停留在 `本单位`。
- 点击 `其他单位` 后，查询区出现 `发送单位` 字段。
- `其他单位` 表格出现 `发送单位` 列，并展示大竹县、达川区等模拟数据。
- 返回 `本单位` 后，恢复原有字段和本单位记录。
