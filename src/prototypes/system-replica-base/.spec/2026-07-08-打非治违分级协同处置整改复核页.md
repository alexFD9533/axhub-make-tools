## 页面定位
本页承接打非治违“分级协同处置”第 1 页，用于回答四个问题：
1. 矿上改没改。
2. 县里过没过。
3. 为什么被退回。
4. 是否需要升级流转。

## 页面结构决策
1. 标题顶条：仅显示页面标题和退出按钮。
2. 工具区：统计周期、煤矿检索、状态筛选、县级视角和重点矿井快捷动作。
3. 顶部指标：待整改、待复核、复核退回、已销号、逾期事项。
4. 焦点摘要区：页面关注点、升级判断摘要、整改前后对比。
5. 主工作区：
   - 左侧为整改复核任务台账；
   - 右侧为复核结论、退回原因排行、升级流转判断。
6. 底部支撑区：附件材料、复核动作提示。
7. 底部流程带：问题下发 -> 企业整改 -> 提交复核 -> 县级复核 -> 销号归档。
