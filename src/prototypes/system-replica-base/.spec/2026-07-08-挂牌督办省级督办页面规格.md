# 挂牌督办 / 省级督办页面规格（2026-07-08）

## 页面定位

- 页面归属：蜀安首页 Av3 drilldown 子页面，位于打非治违“分级协同处置”第 4 页。
- 页面职责：承接重大事项督办场景，集中回答责任压到谁、时限到哪、催办几次、能否销号。
- 实现边界：仅实现 `DrilldownFrame` 的 lower `drill-stage` 内容区，不复制系统顶栏、首页导航或全局工具区。

## 页面结构

1. 页面标题区
   - 标题：`挂牌督办 / 省级督办`
   - 副标题：`蜀安首页 Av3 / 打非治违 / 分级协同处置第4页`
   - 右侧仅保留 `退出` 按钮

2. 内容区工具栏
   - 统计周期：近30天
   - 区域筛选：全部区域
   - 状态筛选：待销号、超期未办
   - 右侧信息：统计口径、更新时间、刷新按钮

3. 顶部指标
   - 挂牌总数
   - 超期未办
   - 催办次数
   - 待销号
   - 已销号

4. 主体区域
   - 左侧：重大事项督办台账 + 销号审核区
   - 右侧：处置闭环时间轴 + 催办记录与领导批示

## 主台账字段

| 字段 | 说明 |
| ---- | ---- |
| 事项编号 | 督办台账主键 |
| 煤矿名称 | 对应被督办煤矿 |
| 挂牌原因 | 对应问题来源与挂牌事由 |
| 责任单位 | 属地监管 + 企业责任主体 |
| 督办时限 | 当前节点的完成截止时间 |
| 当前节点 | 下发、签收、整改反馈、复核、销号中的当前阶段 |
| 状态 | 正常推进、临期催办、超期未办、待销号、再整改等 |

## 详情与审核区

### 处置闭环时间轴

- 节点顺序：下发、签收、整改反馈、复核、销号归档
- 每个节点展示时间和当前动作说明

### 催办与批示

- 催办记录：时间、渠道、催办对象
- 通报状态：已通报未销号 / 未通报 / 已闭环等
- 领导批示：强调责任压实、逾期说明和销号审慎原则
- 销号判断：通过状态标签和条件清单表达是否可销号

### 销号审核区

- 销号申请：申请事项、申请单位、提交时间、佐证材料
- 审核结论：通过 / 退回补正 / 不通过
- 退回与再整改：退回原因、再整改时限、再整改要求

## 视觉与布局要求

- 延续 `system-replica-base` 现有深蓝指挥舱风格
- 面板使用深蓝渐变底、亮蓝描边、浅蓝辅助文字、红橙青绿状态高亮
- 页面顶部不再出现首页全局导航的重复表达
- 所有筛选、台账、时间轴和审核区均收纳在内容舞台内
