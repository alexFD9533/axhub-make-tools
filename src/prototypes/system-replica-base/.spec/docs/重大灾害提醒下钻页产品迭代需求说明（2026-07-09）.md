# 重大灾害提醒下钻页产品迭代需求说明（2026-07-09）

## 一、概要

重大灾害提醒下钻页是蜀安首页 V3「日常监管 - 重大灾害提醒」模块的二级页面，用于展示全省范围内重大灾害事件对煤矿的影响，并支持撤人处置跟踪。

本次迭代围绕撤人业务状态流转、时间线交互和特殊情况处理进行收敛。

## 二、页面定位

| 项目 | 说明 |
|------|------|
| 页面路由 | `shuan-home-command-v3-major-hazard-reminder` |
| 入口来源 | 蜀安首页 V3 → 日常监管模块 → 重大灾害提醒 |
| 页面性质 | 二级下钻页，事件驱动型实时处置工作台 |
| 核心目标 | 跟踪灾害影响煤矿的人员撤人处置状态 |

## 三、业务规则

### 3.1 正常撤人流程

```
监测中 → 通知撤人 → 撤人中 → 疑似完成 → 撤离完成
```

| 状态 | 说明 | 触发条件 |
|------|------|---------|
| 监测中 | 默认状态，系统推荐但未确认 | 灾害接入 |
| 通知撤人 | 监控人员电话通知煤矿撤人 | 点击"通知撤人"按钮并录入通知信息 |
| 撤人中 | 已通知，人员定位跟踪中 | 通知后自动进入 |
| 疑似完成 | 井下人数=0，待确认 | 系统自动判定 |
| 撤离完成 | 监控人员人工确认 | 点击"确认完成"按钮 |

### 3.2 特殊情况：煤矿自行完成撤人

**场景描述：**
煤矿在监测中状态下，未接到通知自行组织人员撤离，井下人数降为0。

**处理流程：**
```
监测中 → [跳过通知撤人节点] → 撤人中 → 疑似完成 → 撤离完成
```

**系统行为：**
1. 检测到井下人数=0时，自动标记"疑似完成"
2. 时间线"通知撤人"节点显示为未完成（----）
3. 监控人员可直接点击"确认完成"，无需先点击"通知撤人"
4. 通知记录可后续补录（点击时间线"通知撤人"节点）

**业务说明：**
- 煤矿自行撤人属于合规行为，系统不强制要求先通知
- 监控人员确认完成后，可后续补录通知记录用于留痕
- 补录仅记录信息，不改变状态流转

### 3.3 状态流转规则

| 当前状态 | 可执行操作 | 下一状态 |
|---------|-----------|---------|
| 监测中 | 通知撤人 / 确认完成 | 撤人中 / 撤离完成 |
| 撤人中 | 无（自动跟踪） | 疑似完成（井下=0） |
| 疑似完成 | 确认完成 | 撤离完成 |
| 撤离完成 | 无（终态） | - |

## 四、页面交互

### 4.1 煤矿卡片状态

| 状态 | 标签颜色 | 操作按钮 |
|------|---------|---------|
| 监测中 | 蓝色 | 通知撤人 + 确认完成 |
| 撤人中 | 橙色 | 无 |
| 疑似完成 | 紫色 | 确认完成 |
| 撤离完成 | 绿色 | 无 |

### 4.2 时间线节点交互

| 节点 | 可点击 | 点击行为 |
|------|--------|---------|
| 监测中 | 否 | - |
| 通知撤人 | 是（监测中/撤人中状态） | 弹出通知录入/展示浮窗 |
| 撤人中 | 否 | - |
| 疑似完成 | 否 | - |
| 撤离完成 | 是（疑似完成/撤离完成状态） | 弹出确认录入/展示浮窗 |

### 4.3 浮窗内容

**通知撤人浮窗：**
- 通知时间
- 执行人
- 对方人员
- 对方电话
- 通话内容

**撤离完成浮窗：**
- 确认时间
- 确认人
- 备注

## 六、边界情况

| 场景 | 处理 |
|------|------|
| 煤矿自行撤人 | 跳过通知节点，直接确认完成 |
| 通知后人员未撤完 | 保持"撤人中"状态，持续跟踪 |
| 井下人数反复变化 | 以最新人员定位数据为准 |
| 多个灾害影响同一煤矿 | 按最新灾害状态展示 |

## 补充：时间线节点交互与视觉反馈

### 点击逻辑

时间线节点不是没填的可以点这里填，而是填过了可以点这里看。

| 节点 | 可点击条件 | 点击行为 |
|------|-----------|---------|
| 监测中 | 不可点击 | - |
| 通知撤人 | 有通知记录时 | 查看已填通知内容 |
| 撤人中 | 不可点击 | - |
| 疑似完成 | 不可点击 | - |
| 撤离完成 | 有确认记录时 | 查看已填确认内容 |

规则说明：
- 节点完成不等于可点击
- 只有已录入数据的节点才能点击查看
- 未录入数据的节点不可点击（即使状态已完成）

### 视觉反馈

| 状态 | 视觉效果 |
|------|---------|
| 有记录可点击 | 节点名称蓝色 + 下划线 + hover 高亮 |
| 无记录不可点击 | 保持默认灰色 |

具体样式：
- 可点击：名称 #5cc6ff 蓝色 + 半透明下划线
- hover：名称 #8ee3ff 变亮 + 下划线变实
- hover：图标背景高亮 + 蓝色光晕
- 不可点击：默认灰色，无下划线，无 hover 效果
