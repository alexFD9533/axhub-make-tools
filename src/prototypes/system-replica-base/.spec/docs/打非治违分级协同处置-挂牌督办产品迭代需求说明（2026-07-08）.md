# 打非治违分级协同处置-挂牌督办产品迭代需求说明（2026-07-08）

## 一、迭代背景

蜀安首页 Av3 右侧打非治违模块已收敛为“五类线索 - 三类专项整治 - 分级处置任务”的三级结构，但分级协同处置链路仍缺少第 4 页“挂牌督办 / 省级督办”承接页面。当前首页只能看到“省挂牌督办”待办数量，无法继续回答责任单位、督办时限、催办记录和销号判断，影响重大事项督办场景的闭环演示。

本次迭代补齐该页面，用于在 drilldown 覆盖层内承接重大事项督办视角，作为打非治违分级协同处置的落地页。

## 二、迭代目标

- 补齐 `shuan-home-command-v3-illegal-disposal-province-supervision` 路由并可直接打开。
- 以重大事项督办为主语义，明确展示责任主体、督办时限、催办次数和销号判断。
- 保持首页 Av3 现有深蓝指挥舱视觉语言，不改动 `DrilldownFrame` 对全局顶部的所有权。
- 让首页“省挂牌督办”任务卡能最小跳转至该页面。

## 三、迭代范围

本次迭代适用于 `src/prototypes/system-replica-base/` 下的蜀安首页 Av3 drilldown 页面。

### 3.1 涉及页面

| 页面 | 迭代类型 | 说明 |
| ---- | -------- | ---- |
| 挂牌督办 / 省级督办 | 新增下钻页 | 新增重大事项督办页面主体内容 |
| 蜀安首页 Av3 右侧打非治违模块 | 最小接入 | “省挂牌督办”任务卡增加跳转能力 |

## 四、需求说明

### 4.1 挂牌督办 / 省级督办

#### 4.1.1 页面定位

- 页面定位：重大事项督办页。
- 页面任务：回答“责任压到谁、时限到哪、催办几次、能否销号”。
- 页面边界：只实现 lower `drill-stage` 内容，不复制系统顶栏、顶部菜单或全局导航。

#### 4.1.2 页面结构

1. 页面标题区
   - 标题：`挂牌督办 / 省级督办`
   - 副标题：`蜀安首页 Av3 / 打非治违 / 分级协同处置第4页`
   - 右侧按钮：`退出`
2. 内容区工具栏
   - 统计周期、区域、状态筛选
   - 统计口径、更新时间、刷新按钮
3. 顶部指标区
   - 挂牌总数
   - 超期未办
   - 催办次数
   - 待销号
   - 已销号
4. 主体区
   - 左侧：重大事项督办台账
   - 右侧：处置闭环时间轴、催办记录、通报状态、领导批示与销号判断
5. 底部审核区
   - 销号申请
   - 审核结论
   - 退回原因
   - 再整改要求

#### 4.1.3 字段与交互

主表格字段如下：

| 字段 | 说明 |
| ---- | ---- |
| 事项编号 | 督办事项唯一编号 |
| 煤矿名称 | 被督办主体 |
| 挂牌原因 | 督办事项来源与挂牌依据 |
| 责任单位 | 属地监管单位与企业责任主体 |
| 督办时限 | 当前整改 / 复核 / 销号截止时间 |
| 当前节点 | 下发、签收、整改反馈、复核、销号归档中的当前阶段 |
| 状态 | 正常推进、临期催办、超期未办、待销号、再整改 |

右侧详情要求如下：

- 处置时间轴按 `下发 -> 签收 -> 整改反馈 -> 复核 -> 销号归档` 顺序展示。
- 催办记录展示时间、渠道和催办对象。
- 通报状态单独展示当前是否已形成省级通报。
- 领导批示区直接展示重大事项督办口径。
- 销号判断区用状态标签和条件清单表达当前是否可销号。

审核区要求如下：

- 销号申请展示申请事项、申请单位、提交时间和佐证材料。
- 审核结论展示通过/退回补正/不通过状态及结论摘要。
- 退回与再整改区展示退回原因、再整改时限和再整改动作清单。

## 五、交互与视觉要求

- 页面仅保留“本页面标题 + 退出按钮”作为顶部表达。
- 所有筛选与工具收纳在内容区工具栏，不上移至全局顶部。
- 视觉延续首页 Av3 深蓝指挥舱语言：深蓝渐变底、亮蓝描边、浅蓝辅助文案、红橙青绿状态高亮。
- 状态标签需能一眼区分“超期未办”“待销号”“再整改”等关键状态。
- 首页“省挂牌督办”任务卡点击后进入该路由；其余任务卡保持现状。

## 六、验收标准

- 直接访问 `#page=shuan-home-command-v3-illegal-disposal-province-supervision` 可打开新页面。
- 页面仅覆盖 `drill-stage` 内容区，不出现重复全局顶栏。
- 顶部指标完整包含挂牌总数、超期未办、催办次数、待销号、已销号。
- 主台账完整展示事项编号、煤矿名称、挂牌原因、责任单位、督办时限、当前节点、状态。
- 右侧详情可回答催办几次、是否已通报、是否可销号。
- 底部审核区完整展示销号申请、审核结论、退回原因、再整改要求。
- 首页“省挂牌督办”卡片可跳转到本页。
