### 2026-06-23（隐蔽面风险业务主线调整）

1. 风险总览页统一采用“煤矿风险等级 + 可疑点 + 核查任务回流”口径：
   
   - 顶部指标改为煤矿总数、高风险煤矿、中风险煤矿、待核查高风险煤矿、逾期核查任务。
   - 高风险煤矿排行字段改为主要可疑点、支撑线索、最近核查状态。
   - 新增可疑点统计区，覆盖停产期间疑似生产、夜间异常活动、同一区域多类异常、数据遮蔽风险、历史问题复发。
   - 检查执行区调整为核查任务与风险回流，强调任务完成不等于风险自动消除。

2. 新增「煤矿风险画像」页面：
   
   - 展示煤矿综合风险分、风险等级、区域排名、支撑线索数、主要可疑点、跟踪状态和回流结论。
   - 展示评分构成、可疑点与检查重点、支撑线索与证据链、最近任务、风险回流判断和检查建议。
   - 页面路径为 `guard-hidden-risk-mine-profile`，可从左侧菜单或总览排行“画像”入口进入。

3. 原型说明面板同步新增煤矿风险画像路由和相关批注说明。

### 2026-06-24（隐蔽面风险评分模型口径修正）

1. 隐蔽面风险业务模型从“可疑点筛选/风险事件”修正为“算法线索基础加权 + 关联增强加权 + 回流修正”。

2. 明确算法与线索关系：
   
   - 算法模型生成线索；
   - 一个算法对同一煤矿可以生成多条线索；
   - 风险强度不使用触发算法数直接表示，而以参与评分线索、线索基础分和关联增强分解释。

3. 风险总览和煤矿风险画像后续改造口径：
   
   - 风险总览保留统计周期、区域、风险等级等主筛选；
   - 可疑点类型、跟踪状态不作为一张图主筛选；
   - 总览增加算法线索态势、关联增强统计和评分构成摘要；
   - 煤矿风险画像增加算法线索贡献、关键线索、关联增强分析、风险解释与核查要点。

4. 同步修正模块级产品迭代需求说明。
