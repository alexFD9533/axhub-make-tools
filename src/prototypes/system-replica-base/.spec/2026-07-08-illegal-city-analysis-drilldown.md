## 页面定位

本页是“复杂事项升级研判页”，核心回答两个问题：

- 县级难以定性的事项，市级如何把瓦斯、通风、人员、产量、用电等多源线索归并到同一事项。
- 市级形成统一意见后，如何明确是继续核查、维持整改、转挂牌督办，还是排除误报。

## 结构决策

页面结构固定为四层：

1. 标题条：页面标题、页面定位文案、退出按钮。
2. 内容工具栏：事项筛选、周期筛选、状态筛选、刷新归并。
3. 核心业务区：
   - 顶部指标条：待研判、研判中、建议升级、退回补查、已形成意见。
   - 左侧主区：复杂事项主表格 + 多源线索归并视图 + 异常时间轴。
   - 右侧意见区：核心争议、研判结论、建议动作、会商记录。
4. 底部动作条：维持整改、继续核查、转挂牌督办、排除误报。

## 接入决策

1. 新增独立页面文件：`IllegalCityAnalysisPage.tsx`。

4. 首页 Av3 “市级研判”卡片做最小接入，点击后打开本页；其他分级处置卡片保持原状。
