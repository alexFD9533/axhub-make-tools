# 2026-06-23 隐蔽面风险综合分析风险总览构建记录

## 背景

基于《PRD-矿数卫士-隐蔽面风险综合分析子模块》，在既有 `system-replica-base` 底座内扩展矿数卫士子系统能力。本模块不新建独立平台，作为矿数卫士下属业务工作区推进。

## 本轮范围

本轮仅构建一级页面「风险总览一张图」，其他页面先登记菜单并保留占位：

- 煤矿风险画像
- 线索证据台账
- 工作安排处置
- 配置管理

## 页面结构

风险总览页面包含：

- 筛选区：统计周期、区域、风险等级、风险类型。
- 顶部指标：煤矿总数、高风险煤矿数、中风险煤矿数、待处置线索数、逾期任务数。
- 区域风险态势：按区县展示煤矿数、高中低风险分布和环比变化。
- 风险变化趋势：展示近 7 天高中低风险煤矿数量变化。
- 高风险煤矿排行：展示煤矿名称、区域、风险分、等级、主要风险类型、关联线索数和环比。
- 检查执行情况：展示任务下发、完成、检查中、整改中、逾期和待复核。

## 设计决策

- 模块入口放在矿数卫士左侧菜单下，名称为「隐蔽面风险综合分析」。
- 页面表达保持综管平台底座风格：顶部查询、指标卡、业务看板、表格排行和任务统计。
- 排行表的「画像」「派任务」先作为路径提示，不在本轮展开真实跳转。
- 其他模块占位页说明页面定位，避免用户误以为已完成全链路。

## 验收重点

- 能从矿数卫士菜单进入「风险总览一张图」。
- 风险总览包含 PRD 中要求的顶部指标、区域风险、煤矿排行、风险趋势和检查执行统计。
- 未展开页面保持占位，不提前铺开全模块。
- 单入口构建通过：`ENTRY_KEY=prototypes/system-replica-base npx vite build`。
