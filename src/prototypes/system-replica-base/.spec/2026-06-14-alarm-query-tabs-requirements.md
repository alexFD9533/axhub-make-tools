## 总体

“报警查询”页不是单一表格。标题下方有 5 个页内标签：

1. 监控报警
2. 人员超时
3. 人员超员
4. 进入限制区域报警
5. 人员求救

本地原型必须在同一 `monitor-alarm-query` 页面中呈现这 5 个标签，并支持点击切换不同筛选区、工具条、表头、数据或空状态。
