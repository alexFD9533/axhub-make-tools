## 组件分解

### 顶部系统栏

- 品牌：圆形“蜀”徽标 + `蜀·安煤矿多源异构监测系统`，标题白色高亮。
- 导航：`首页 / 日常监管 / 打非治违 / 从业人员管理 / 生产管理 / 系统设置`，当前项为亮蓝色斜切块。
- 工具区：视频、通知、用户、设置，保持圆形图标按钮，不使用文字按钮替代。

### 左侧数据栏

- 煤矿生产状态统计：总数在标题右侧，五类状态横向等分；长标题可换行但数字不能被压缩。
- 煤矿生产情况：四张指标卡，数字与单位同一视觉层级，标签另起一行。
- 从业人员情况：四张指标卡，结构同生产情况。
- 数据汇聚情况：六张数据卡横向等分，图标、数量、名称三层。
- 瓦斯/水害：双列并排，图表和列表互不遮挡。

### 中央地图

- 背景：项目已有卫星底图全容器铺底，不能只显示在四川轮廓内部。
- 行政区：项目已有 `sichuan-map.svg` 作为半透明蓝色覆盖，不得 `scaleY` 拉伸。
- 搜索框：位于地图顶部，图标、输入、按钮三列，文字不裁切。
- 点位：圆形蓝色气泡 + 黑色城市标签，标签不压住主数字。
- 图例：左下角半透明卡片，项目可读优先。

### 右侧日常监管

日常监管必须按四段垂直排列：

1. 汇总条：预警总数、分级分布、处置率。
2. 报警分类：环境监测、人员定位、水害防治三张卡。
3. 业务入口：危险作业报备、风险分级管控、隐患排查三张卡。
4. 重点提醒：证照到期提醒、重大灾害提醒两张横向卡。

硬约束：

- 第二段和第三段不得重叠。
- 每张卡的标题、总数、两条明细必须拥有独立行高。
- 右上角时间筛选只占标题区，不挤占内容区。

### 右侧打非治违

- 五类线索源：瓦斯、风、人、煤、电，横向五卡。
- 三类专项：三张并列卡。
- 四级处置：煤矿整改、县级走访、市级研判、省挂牌督办四卡。
- 底部不得被视口裁切；如高度不足，优先压缩卡内留白，不牺牲文字可读性。
