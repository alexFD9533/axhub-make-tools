# 煤矿基础数据使用规则

## 数据来源

- 原始文件：`src/resources/shuan/四川省应急厅确认在册174家煤矿名单.xlsx`
- 代码数据：`src/resources/shuan/sichuan-coal-mines-2026.json`
- 数据口径：2026 年初全省煤矿清单，共 174 家。

## 使用规则

1. 后续蜀安原型中涉及煤矿名称、所属市州、所属区县、矿井类型、最新状态、瓦斯等级、水文地质类型、生产能力等基础信息时，优先使用 `sichuan-coal-mines-2026.json`。
2. 不再随手编造煤矿名称。若页面只是展示局部样例，应从 JSON 中按市州、区县、矿井类型或状态筛选后取样。
3. 页面统计口径需要说明范围：全省口径使用 `total = 174`；市州口径使用 `cityCounts` 中对应市州数量，例如达州市为 41 家。
4. 如果业务演示必须临时使用非名单数据，应在代码或文档中标注为“演示占位”，后续拿到真实资料后替换。
5. 原始 Excel 是第一信源；如后续用户提供更新版名单，应先替换原始文件，再重新生成 JSON，避免多处手工维护。
